Telco Account Team Performance Dashboards

Overall Goal

    This project aims to enhance the performance of beginner experts handling a telco account through data-driven decisions. Utilizing a PowerBI dashboard, it provides actionable recommendations based on KPIs, enabling coaches to identify areas for improvement, offer targeted training, and boost team performance and NPS. The project also empowers beginner experts by providing valuable feedback to achieve incentives and contribute to team success.

Data Gathering and Preparation

  • Engaged in a brainstorming session with the supervisor to determine essential data for enhancing expert and team performance.
  • Collaborated with the supervisor to develop a data collection plan involving listening to each expert's calls and using a checklist to identify key behaviors. The collected data is submitted for weekly analysis.
  • The data is prepared, cleaned, and aggregated in Excel to ensure accuracy and reliability before being incorporated into the PowerBI dashboard.

Significance

  • The project showcases the effectiveness of data-driven decisions in improving team performance and increasing NPS.
  • The use of a PowerBI dashboard allows for easy tracking and analysis of key metrics and KPIs.
  • The project demonstrates proficiency in data analysis, project management, collaboration, data preparation, and data visualization skills, valuable assets in the data industry.

Are data analysts undervalued in the Philippines?

Overall Goal

    Conduct a comprehensive investigation into the compensation and potential undervaluation of data analysts in the Philippines using Python. Utilize powerful packages like pandas and matplotlib to analyze and visualize the data, providing valuable insights into salary trends and industry dynamics.

Data Gathering and Preparation

  • Developed a custom web scraper using Python and Selenium to extract data from job postings.
  • Created a Python program to clean and preprocess the scraped data automatically, saving it in CSV format.

Significance

  • Provides valuable insights for data analysts and employers in the industry.
  • Raise awareness of potential undervaluation issues.
  • Utilizes self-developed tools to enhance data reliability and analysis.
